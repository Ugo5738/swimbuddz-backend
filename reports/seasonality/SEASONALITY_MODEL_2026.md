## How to Read This Model

### The Formula

```
Expected Demand = Baseline × Seasonal Index × Trend × Campaign Multiplier
```

- **Baseline** = average monthly demand with seasonality removed
- **Seasonal Index** = month-specific multiplier (1.0 = average month)
- **Trend** = compound growth since launch
- **Campaign Multiplier** = manual boost for known events/promos

### Status Labels

| Status                | Meaning                                   | Action                                               |
| --------------------- | ----------------------------------------- | ---------------------------------------------------- |
| Expected Seasonal Dip | Demand is _supposed_ to be low this month | Don't panic. Run retention, not acquisition.         |
| On Track              | Actual demand matches forecast            | Continue as planned.                                 |
| Outperforming         | Actual demand exceeds upper bound         | Investigate what worked — double down.               |
| Underperforming       | Actual demand below lower bound           | Investigate. Is it operational? Market? Competition? |

### Decision Rules

| Situation                              | Action                                                   |
| -------------------------------------- | -------------------------------------------------------- |
| Rainy season dip (Jun–Aug)             | Normal. Focus on retention and indoor options.           |
| Jan–Feb below forecast                 | Investigate — these should be peak months.               |
| Two consecutive months underperforming | Operational review needed.                               |
| School holiday + low demand            | Expected. Use for planning and content.                  |
| Month-end spike                        | Normal salary-cycle effect. Schedule promos accordingly. |
